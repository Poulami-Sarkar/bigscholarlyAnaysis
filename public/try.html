About

This website is to analyse the scholarly ,publication of the data comprising of close to millions of research publications, authors, citations and publication venues. The data is present in multiple tables, hosted in MySQL Cloud service. In this project we intend to build a web based application that presents various insights drawn from this dataset as visual network, charts, tables and heatmaps.
